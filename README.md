# ☕ Café Junction

Café Junction is a café website made using HTML and Tailwind CSS.  
I created this project to practice Tailwind CSS and improve my frontend development skills by building a real-world café landing page.

The website has a modern café design with coffee menu cards, snacks section, customer reviews, and a clean UI.

---

# 🚀 Features

- Responsive Navbar
- Hero Section
- Coffee Menu Cards
- Snacks Section
- About Section
- Customer Reviews
- Footer Section
- Hover Effects
- Modern UI Design

---

# 🛠️ Technologies Used

- HTML5
- Tailwind CSS

---

# 📂 Sections Included

- Home
- Popular Menu
- Snacks & Bakery
- About Café Junction
- Reviews
- Footer

---

# 🎨 Design

I used dark coffee-style colors and warm tones to give the website a premium café feel.

Main colors used:
- Dark Brown
- Coffee Brown
- Cream Text Colors
- Amber Highlights
